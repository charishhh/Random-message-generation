# Ransom Note Generator

A web application that generates stylized ransom notes with magazine-style cut-out letters. Each letter appears to be cut from different sources with various fonts, colors, sizes, and rotations.

## Usage

1. Open `index.html` in a web browser
2. Type your message in the text input field
3. Watch as your text is transformed into a ransom note style
4. Click "Send note as URL" to get a shareable link
5. Click "Image" to download your ransom note as an image

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts
- html2canvas (for image generation)

## How It Works

The application:
- Takes user input text
- Splits it into individual letters
- Applies random fonts from a curated collection
- Adds random rotations and size variations
- Applies vibrant colors and patterns
- Renders everything with magazine cutout-style effects

## Browser Support

Works on all modern browsers that support:
- CSS3 features (transforms, gradients)
- ES6 JavaScript
- Canvas API (for image export)
